## HR Analytics Dashboard (Tableau Project)

### Tech Stack
- Tableau  
- SQL  
- Microsoft Excel  
- Data Visualization  
- HR Analytics  

### Project Overview
This project focuses on analyzing human resource data to uncover insights related to employee attrition, performance, and overall workforce distribution. Using Tableau, an interactive dashboard was designed to transform raw HR data into meaningful visual representations that help HR teams and management make informed decisions.

### Key Features
- Employee attrition analysis based on department, job role, age group, and experience
- Performance and salary trend analysis across different roles and levels
- Workforce diversity insights including gender distribution and department-wise breakdown
- Interactive filters for department, education, experience, and job role
- KPI cards highlighting total employees, attrition rate, average salary, and job satisfaction

### Data Processing
- Cleaned and structured raw HR data using Microsoft Excel
- Used SQL queries to filter, aggregate, and prepare data for visualization
- Ensured data consistency and accuracy before dashboard creation

### Outcomes & Insights
- Identified high-attrition departments and roles requiring strategic intervention
- Revealed patterns between experience, salary, and employee retention
- Enabled HR stakeholders to quickly explore trends through interactive visuals

### Use Case
This dashboard can be used by HR managers and decision-makers to:
- Improve employee retention strategies  
- Optimize hiring and workforce planning  
- Monitor HR KPIs in real time  
- Support data-driven organizational policies
